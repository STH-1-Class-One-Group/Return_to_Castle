# 🏺 RTCW Archive App
> **Return to Castle Wolfenstein: The Digital Preservation Project**

이 프로젝트는 2001년 작 FPS 명작 **'Return to Castle Wolfenstein(RtCW)'**의 게임 요소, 설정 파일, 모드(Mods), 그리고 역사적 기록들을 체계적으로 보존하고 전시하기 위한 **아카이브 애플리케이션**입니다.

---

## 🏛️ 아카이브 주요 내용
* **Mission Logs:** 싱글 플레이어 에피소드 및 스토리 기록
* **Arsenal:** 무기 데이터, 데미지 테이블 및 개발 비화
* **Bestiary:** 오컬트 크리처 및 적 병사들의 속성 아카이브
* **Technical Docs:** `autoexec.cfg` 설정 가이드 및 퀘이크 3 엔진 최적화 기록
* **Community Heritage:** 전설적인 멀티플레이어 맵과 모드 데이터베이스

---

## 🛠️ 개발 스택 (Current)
* **Environment:** Git / GitHub
* **Core:** (여기에 React, Next.js, 혹은 단순 HTML/JS 등 사용 기술 기입)
* **Database:** JSON 기반 정적 데이터 아카이빙

---

## 📂 프로젝트 구조
```text
/rtcw-archive-app
├── /assets          # 이미지, 사운드 샘플, 아이콘
├── /data            # 게임 데이터 JSON (무기, 적군 정보 등)
├── /docs            # 게임 메뉴얼 및 텍스트 아카이브
├── index.html       # 메인 아카이브 인터페이스
└── README.md        # 프로젝트 명세서

```
