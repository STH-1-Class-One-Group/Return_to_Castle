# 🏰 Return to Castle Wolfenstein (RtCW) Review Page

![RtCW Banner](https://img.shields.io/badge/Game-RtCW-red?style=for-the-badge)
![Category-Review-blue?style=for-the-badge](https://img.shields.io/badge/Category-Review-blue?style=for-the-badge)

> **"잠입과 액션, 그리고 오컬트의 완벽한 조화"** > 2001년 출시되어 FPS 장르에 한 획을 그은 명작, *Return to Castle Wolfenstein*의 리뷰 웹페이지 프로젝트입니다.

---

## 🎮 프로젝트 개요
이 프로젝트는 고전 FPS 게임인 **Return to Castle Wolfenstein**의 게임 플레이, 그래픽, 사운드 및 역사적 가치를 분석하고 기록하기 위한 리뷰 웹사이트입니다. 

### 주요 특징
* **반응형 디자인:** 데스크탑과 모바일 어디서든 쾌적한 리뷰 감상 가능
* **인터랙티브 요소:** 게임 내 무기 정보 및 적 캐릭터 도감 확인
* **멀티미디어 지원:** 게임 스크린샷 및 관련 영상 임베딩

---

## 🛠 사용 기술 (Tech Stack)
* **Frontend:** HTML5, CSS3, JavaScript (또는 React/Vue 등 사용하시는 기술로 수정하세요)
* **Design:** Retro & Dark Theme (Wolfenstein 테마)
* **Deployment:** GitHub Pages

---

## 📝 리뷰 섹션 구성
1. **Introduction:** 게임의 배경 스토리 및 주인공 'B.J. 블라즈코윅즈' 소개
2. **Gameplay:** 잠입 요소와 화끈한 슈팅 액션의 밸런스 분석
3. **Graphics & Sound:** 당시 파격적이었던 퀘이크 3 엔진의 성능과 몰입감 넘치는 사운드
4. **Conclusion:** 20년이 지난 지금도 플레이할 가치가 있는가?
